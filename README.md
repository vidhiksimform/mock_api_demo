# Mocking HTTP request using MSW

Created a small demo on how to mock api using MSW.For demo purpose I have created todolist app for fetching todos and post todos.

# Installation and setup instruction
Installation : 
 ` npm install`

To run locally :
 `npm run dev`

Visit app : 
 http://localhost:5173

# How to run testcase
`npm run test`
